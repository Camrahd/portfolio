# Dharmendra Reddy Chitte — Portfolio

Single-page portfolio built with React + Vite. Warm paper-and-ink editorial design: Fraunces + Inter, three-color system (paper / ink / rust), scroll-triggered reveals, and a canvas particle field in the hero.

## Run

```sh
npm install
npm run dev      # http://localhost:5173
npm run build    # production build in dist/
```
