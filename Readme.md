# Dharmendra Reddy Chitte — Portfolio

Single-page portfolio built with React + Vite. Warm paper-and-ink editorial design: Fraunces + Inter, three-color system (paper / ink / rust), scroll-triggered reveals, and a canvas particle field in the hero.

## Run

```sh
npm install
npm run dev      # http://localhost:5173
npm run build    # production build in dist/
```

## Before deploying

- Drop your resume at `public/resume.pdf` (the hero "View Resume" button points there).
- Replace the publication `href`s in `src/App.jsx` with the real DOI / IEEE Xplore links.
- Confirm the project GitHub repo slugs in `src/App.jsx` match your actual repos.
